Для запуска проекта необходимо:
1) Установить все пакеты из файла requrements.txt
2) ДОБАВИТЬ В ФАЙЛ .env ТОКЕН ДЛЯ ТЕЛЕГРАМ БОТА
2) Запустить файл main.py и не останавливать программу во время взаимодействия с ботом.

Бот умеет отвечать на команду /help, в которой описывается способ взаимодействия с ним:

/start - активируем бота
/problem - запрашиваем у бота задачу
/statistic - запрашиваем количество решенных задач за все время
/rating - рейтинг среди всех пользователей
/rate x - оценить бота рейтингом x :)
Для ответа на задачу нужно просто написать ответ боту.

Между запусками скрипта программа сохраняет статистику всех пользователей. В начале подгружает ее ( load_data, save_data)
